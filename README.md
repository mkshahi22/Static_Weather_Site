# Shahi Weather App

A simple static weather website built using HTML, CSS, and JavaScript. The app fetches real-time weather data using the OpenWeatherMap API and displays key weather parameters for a selected location.

## Features
- Current temperature, humidity, and weather condition
- Weather icon based on real-time data
- City-based weather search
- Responsive design for mobile and desktop

## Setup Instructions
1. Clone the repository  
   ```bash
   git clone https://github.com/mkshahi22/Static_Weather_Site.git
   ```

2. Open `index.html` in your browser or launch with Live Server (VS Code extension)

3. Update your API key from [OpenWeatherMap](https://openweathermap.org/api) inside the JavaScript file:
   ```javascript
   const apiKey = "YOUR_API_KEY_HERE";
   ```
